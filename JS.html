JavaScript itu apa sih??


Bayangin deh kamu lagi main sama robot yang bisa ngobrol juga ngelakuin semua perintahmu. Nah, JavaScript itu kayak otak yang kasih tau halaman web buat ngelakuin hal-hal keren!

JavaScript

1. Aksi (Actions): JavaScript ngebuat kamu kasih tau halaman web untuk ngelakuin banyak hal. Misalnya, kamu bisa buat tombol yang pas dipencet bisa ngeubah warna halaman atau menampilin pesan.


2. Mengubah Isi (Changing Content): Kamu bisa pake JavaScript buat ngubah teks atau gambar di halaman tanpa perlu me-refresh semua halaman. Ini cocok banget buat cerita yang bisa berubah-ubah sesuai keinginan!


3. Berinteraksi (Interaction): JavaScript ngebuat halaman web bisa interaksi sama kamu. Kamu bisa membuat kotak teks untuk menulis pesan atau game yang kamu mainkan di halaman web.

JavaScript

4. Pengulangan (Loops): Bayangin deh ini kayak main tapi ngulang-ulang. Kamu bisa bilang ke JavaScript buat ngelakuin sesuatu secara berulang kali, kayak ngitung dari 1 sampe 10.


5. Keputusan (Decisions): Ini kayak buat pilihan. Kamu bisa kasih tau JavaScript buat ngelakuin hal tertentu jika suatu kondisi terpenuhi. Contohnya, kalo kamu punya lebih dari 5 kelereng, kamu mau main.


6. Permainan (Games): Kamu bisa buat permainan sederhana pake JavaScript, kayak game menebak angka atau game memotong tali.


7. Animasi (Animation): Kamu bisa membuat elemen di halaman webmu gerak juga berubah-ubah. Ini kayak membuat robot bergerak di layar!

Robot



Jadi, JavaScript itu temen ajaib yang kasih kekuatan pada halaman webmu untuk bergerak, berbicara, dan berinteraksi sama kamu. Ini kayak ngasih hidup ke robot atau mainan favoritmu di dunia digital!
